🎓 Student Performance Dashboard | Power BI

Transforming educational data into actionable insights through interactive visualizations and analytics.

📖 Overview

The Student Performance Dashboard is an interactive Business Intelligence solution developed using Microsoft Power BI to analyze academic performance, attendance patterns, and student achievement metrics.

This dashboard transforms raw educational data into meaningful insights, enabling educators and administrators to monitor performance trends, identify learning gaps, and support data-driven decision-making.

🎯 Project Objectives

✔ Analyze overall student performance

✔ Monitor attendance and engagement

✔ Compare subject-wise academic achievement

✔ Identify top and low-performing students

✔ Visualize grade distribution patterns

✔ Generate actionable educational insights
