# 🎓 **Student Performance Dashboard | Power BI**

## 📖 **Overview**

The Student Performance Dashboard is an interactive Business Intelligence project developed using Microsoft Power BI to analyze academic performance, attendance patterns, and student achievement metrics.

---

## 🎯 **Project Objectives**

- Analyze overall student performance
- Monitor attendance and engagement
- Compare subject-wise academic achievement
- Identify top and low-performing students
- Visualize grade distribution patterns

---

## 📊 **Dashboard Features**

### 📌 **Key Performance Indicators (KPIs)**

- Total Students
- Average Score
- Highest Score
- Lowest Score
- Attendance Percentage
- Pass Percentage

### 📈 **Interactive Visualizations**

- Subject-wise Performance Analysis
- Student Score Distribution
- Attendance Analysis
- Grade Distribution
- Gender-based Performance Comparison
- Top Performing Students
